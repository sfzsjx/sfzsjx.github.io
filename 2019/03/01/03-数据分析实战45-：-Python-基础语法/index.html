Python 基础语法

Python 语言特点

Python 语言最大的优点就是简洁，同时有大量的第三方库，功能强大，能够解决数据分析的大部分问题。比如科学计算工具NumPy 和 Pandas库，深度学习工具 Keras 和 TensorFlow,以及机器学习工具 Scikit-learn。

安装及 IDE 环境

安装Python 3.x 官网下载安装。
Python IDE 推荐使用 PythonCharm
