一、蓝牙信标信号强度转换

计算公式： d = 10^((abs(RSSI) - A) / (10 * n))
其中：
d - 计算所得距离
RSSI - 接收信号强度（负值）
A - 发射端和接收端相隔1米时的信号强度
n - 环境衰减因子
