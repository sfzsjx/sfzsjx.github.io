jupyter 安装

2019-01-21
python

选择管理员运行cmd,执行以下命令：

 pip install jupyter

进入目录 C:\Program Files\Python37\Scripts>，运行命令：

 cd C:\Program Files\Python37\Scripts             
 jupyter notebook

注意管理员权限问题。

 http://localhost:8888/tree

 pip install numpy
 pip install -U scikit-learn
 pip install -U pandasql
 pip install graphviz
 pip install pydotplus
