Linux 常用命令

linux系统中清空文件内容的三种方法

1.使用vi/vim命令打开文件后，输入”%d”清空，后保存即可。但当文件内容较大时，处理较慢，命令如下：

vim file_name
:%d
:wq

2.使用cat命令情况，命令如下：

cat /dev/null > file_name

3.使用echo命令清空，此时会在文件中写入一个空行“\n”，命令如下：

echo “”>file_name
